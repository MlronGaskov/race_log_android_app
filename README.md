# RaceLog Android

Android-приложение на Kotlin + Jetpack Compose (шаблон Empty Activity). На данный момент реализованна авторизация и навигация. Вместо страниц "Профиль", "Группы", "Результаты" стоят заглушки.

## Стек
- Kotlin
- Jetpack Compose
- Navigation Compose
- Retrofit + OkHttp
- Kotlinx Serialization
- DataStore (хранение токенов)
